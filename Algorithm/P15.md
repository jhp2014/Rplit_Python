+ 파라메트릭 서치
    + 원하는 조건을 만족하는 가장 알맞은 값
    + 보통 이진탐색을 사용한다.
    + while 문 사용시, 현재로서 가장 최선의 값을 저장 하고 있어야 한다.
+ **정렬**을 활용 할 줄 알아야 한다. 
```python
# 내 실수

while s <= e:
    if target >= mid:
        s = mid
    # 최선의 값을 유지하기 위해 이렇게 코딩했다.
    #하지만 이렇게 작성하면 마지막에 while 문 못 빠져나온다.


while left <= right:
    if total >= m:
        result = mid    #최선의 값을 가지고 있고
        left = mid + 1    #이진탐색
```
+ while 범위 설정하는거 좀 헷갈리네
